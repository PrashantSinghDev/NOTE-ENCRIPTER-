This project was created to provide a more secure way to store personal information than a regular notepad on a mobile phone or Windows. It works as a personal vault where users can safely manage their notes.

The program provides five main options:

* **Add** – Save new encrypted notes.
* **View** – Display all saved notes.
* **Decrypt** – Decrypt and read encrypted notes.
* **Delete** – Remove unwanted notes.
* **Exit** – Close the application safely.

The main objective of this project is to protect sensitive information by keeping it encrypted, making it more secure than storing plain text in a standard notepad application.
